## Proto Node Labs

A small builder studio shipping client products, internal tools, and AI-integrated workflows. We work close to the operator and build for real use, not demos.

---

### What we build

- **Client web apps and sites** — production builds on Vercel and Cloudflare, backed by Neon
- **Internal tools** — operator-facing dashboards, lead-response automation, workforce management
- **AI-integrated products** — assistants with custom knowledge sources, rate limiting, and abuse safeguards baked in
- **Workflow systems** — persistent context, repeatable patterns, opinionated defaults

---

### How we work

We test across the major AI platforms — OpenAI, Google Gemini, Claude, Code Puppy, GLM — and build with what fits the use case. Not a single-platform shop. Production ships are reviewed before merge; quality bar is enforced through branch protection, not vibes.
